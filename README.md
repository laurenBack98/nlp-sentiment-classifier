Sentiment analysis using Natural Language Processing (NLP) techniques on product/service reviews.

## 📋 Description

This project performs a comprehensive sentiment analysis on a dataset of reviews. Using advanced NLP techniques, it classifies reviews into sentiment categories (positive, negative, neutral) and provides insights into opinion patterns.

### What's it for?
- Automatically classify sentiments in texts
- Identify opinion patterns in large volumes of data
- Generate reports and analysis visualizations
- Understand user/customer perception

## ✨ Key Features

- ✅ Text cleaning and preprocessing
- ✅ Exploratory Data Analysis (EDA)
- ✅ Sentiment Classification (Positive, Negative)
- ✅ Statistical visualizations and graphs
- ✅ Model performance metrics
- ✅ Results export

## 📦 Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
